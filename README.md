# Parte 4 - Configurar el Backend de Correo y Modificar Info_request

En esta parte, se ha configurado el backend de correo y se ha modificado la vista Info_request para enviar un correo electrónico cuando se recibe una solicitud de información. Además, se ha creado una plantilla de éxito para mostrar un mensaje de confirmación al usuario.

Pasos Realizados:

	1.- Se ha añadido la configuración necesaria para el backend de correo.

    2.- Se ha modificado la vista para enviar un correo al recibir una solicitud de información.

	3.- Se ha creado una plantilla HTML para mostrar un mensaje de éxito.

 	4.- Se ha añadido una nueva ruta que apunta a la vista de éxito.
    
    5.- Se ha creado la vista info_request_success que renderiza la plantilla de éxito.

Validación:

	El sistema envía correos correctamente al recibir solicitudes de información y muestra un mensaje de éxito al usuario.
